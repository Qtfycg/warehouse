# 系统架构说明
## 各模块功能概述
```text
1.user
    - 用户注册、登录、信息修改等功能
2.admin
    - 管理员管理用户、商品、订单等功能
3.product
    - 商品管理、商品分类、商品搜索等功能
4.notify
    - 消息通知、邮件发送等功能
5.order
    - 订单管理、订单查询、订单支付等功能
6.marketing
    - 营销活动、优惠券、积分等功能
7.common
    - 公共模块，包含工具类、配置类等
8.gateway
    - 网关模块，负责请求路由、负载均衡、安全认证等功能
9.file
    - 文件上传、下载、存储等功能
10.geo
    - 地理位置服务，提供地址解析、坐标转换等功能
11.payment
    - 支付模块，处理支付请求、支付结果通知等功能
12.search
    - 搜索服务，提供商品搜索、推荐等功能
13.stock
    - 库存管理，处理商品库存的增减、查询等功能
14.registry
    - 服务注册与发现，管理微服务实例的注册和查询
15.config
    - 配置中心，集中管理系统配置，支持动态配置更新
```
