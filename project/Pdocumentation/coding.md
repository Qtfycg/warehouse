# coding

```
是一个面向开发者的技术内容分享社区.
可以在平台上发布技术文章、分享代码片段、参与技术问答，帮助彼此快速成长。
```

### 技术栈
```
Spring Boot、Vue、MySQL、Redis、MyBatis-Plus、Element-UI、Quil、ElasticSearch、Docker、
```

### 主要功能
```
1.内容分享
  1.文章编辑器
  2.文章分类及标签
  3.点赞、评论、收藏
  4.个性化推荐（未来扩展）
2.技术讨论
  1.技术问答
  2.话题互动
3.用户社交
  1.个人主页
  2.关注系统
  3.积分体系
4.代码分享
  1.在线编辑器
  2.收藏，分享
5.后台管理
  1.文章/内容审核
  2.用户管理
  3.数据统计
6.未来扩展
  1.付费专栏
  2.用户合作
```
