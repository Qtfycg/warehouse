# 开发规范（命名、分支、提交等）
## 1. 通用状态码

~~~text
1.用户模块
    1.200--成功
    2.300--失败
    3.301--用户权限不足
    4.302--用户已存在
    5.303--用户不存在
    6.304--密码错误、验证码错误
2.管理员模块
    1.400--成功
    2.500--失败
    3.501--管理员权限不足、管理员不存在
    4.502--非管理员账号
    5.503--密码错误、验证码错误
3.产品模块
    1.600--成功
    2.700--失败
    3.701--修改产品信息失败
    4.702--产品不存在


~~~
